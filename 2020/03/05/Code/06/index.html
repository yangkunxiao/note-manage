什么是函数柯里化？

函数柯里化就是使函数先接受一部分参数，使用闭包缓存已接受的函数，返回一个新的函数，在新的函数中接受剩余参数。常常用作实现参数复用、尽量使代码函数化，增加可读性。

实现函数柯里化
